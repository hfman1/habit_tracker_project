# Habit Tracker Pro

Консольное + GUI приложение для отслеживания привычек.

## Возможности
- Добавление/удаление привычек.
- Отметка выполнения.
- Статистика (streak, progress).
- Визуализация в GUI (matplotlib).
- Мотивационные сообщения.

## Установка
```bash
git clone <repo-url>
cd habit_tracker
python -m pip install -r requirements.txt
